🎤 STEP 1 — Audio Transcription

Python Example
from openai import OpenAI
client = OpenAI()

audio = open("consultation.wav", "rb")

transcript = client.audio.transcriptions.create(
    model="whisper-1",   # OpenAI Whisper model
    file=audio
)

TRANSCRIPT_TEXT = transcript.text
