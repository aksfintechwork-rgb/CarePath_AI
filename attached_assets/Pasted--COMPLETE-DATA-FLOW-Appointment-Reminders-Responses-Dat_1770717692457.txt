🔹 STEP 2: WHATSAPP SCHEDULER (CRON / WORKER)
Trigger

Runs every 1 minute

SYSTEM PROMPT — SCHEDULER
Every minute:

1. Fetch all care_events where:
   - event_type = 'medicine'
   - status = 'pending'
   - scheduled_time <= now()

2. For each event:
   - Validate visit is still active
   - Validate patient has whatsapp_number
   - Send WhatsApp reminder

DB QUERY
SELECT
  ce.id AS care_event_id,
  ce.scheduled_time,
  p.whatsapp_number,
  p.name AS patient_name,
  m.name AS medicine_name
FROM care_events ce
JOIN visits v ON v.id = ce.visit_id
JOIN patients p ON p.id = v.patient_id
JOIN medicines m ON m.visit_id = v.id
WHERE ce.status = 'pending'
AND ce.scheduled_time <= NOW();
